# Exemples

Ce dossier accueillera des exemples anonymisés issus de vrais appels d'offres traités avec ce skill. Vide pour le moment.

## Vocation

Au fil des utilisations, capitaliser des cas représentatifs :

- AO mono-lot simple
- AO multi-lot
- AO avec compléments client (illustration de la logique de mise à jour)
- AO avec scoring Go/NoGo

Pour chaque exemple, prévoir :

- Une description du contexte (anonymisé)
- Les livrables produits (synthèse, plan, Go/NoGo)
- Le retour d'expérience après dépôt et instance Go/NoGo

## Règles d'anonymisation avant intégration

- Aucun nom de client réel.
- Aucun montant réel (remplacer par des fourchettes).
- Aucun nom de personne (interne ou externe).
- Aucune référence projet réelle.
- Aucun document client (uniquement les livrables produits par le skill).
